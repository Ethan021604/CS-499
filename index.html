About the Project

A previous project, rebuilt with stronger purpose.

The artifact selected for this ePortfolio is a mobile Weight Tracker application originally created during my computer science program. The application allows a user to log in, create and manage weight entries, and track progress toward a goal.

For my capstone, I selected this application because it gives me a clear way to show how my skills have developed. Rather than leaving the project in its original form, I am improving its design, functionality, and data handling to create a stronger final artifact.

Capstone Roadmap

Enhancement Areas

Each enhancement focuses on a different area of computer science while contributing to one complete mobile application.

01 Software Design

Software Engineering and Design

Rebuilding the original Kotlin artifact in Java while improving project organization, readability, maintainability, and interface structure.

  • Kotlin to Java conversion
  • Improved code organization
  • Cleaner UI structure
02 Algorithms

Algorithms and Data Structures

Expanding the application with meaningful calculations and improved data processing for recorded weight entries.

  • Total weight gained or lost
  • Highest and lowest weight entries
  • Progress toward goal weight
03 Databases

Databases and Security

Improving login handling, input validation, database operations, and defensive programming practices throughout the application.

  • Safer login validation
  • Improved CRUD handling
  • Security-focused data management
